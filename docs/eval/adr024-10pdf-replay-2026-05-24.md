# ADR 024 10-PDF Replay — 2026-05-24

## Scope

Replayed the original 10-PDF production pilot after ADR 024 single-choice answer
sanitization.

This was a dry-run replay using existing MinerU artifacts with `--resume`, so
MinerU parsing, database writes, asset persistence, crop, pHash, duplicate
candidates, and visual candidates were intentionally skipped. The replay
isolates Layout Ownership, real DeepSeek structuring, and quality hardening.

## Verdict

PASS for ADR 024 quality objective.

The targeted warning class, `answer_not_in_choices`, dropped from 7 to 0 across
the full 10-PDF pilot replay. Overall quality warnings dropped from 9 to 2
without introducing failed questions.

## Remaining Issue

The only remaining quality warning class is `too_few_choices`.

Next recommended hardening target: inspect the two affected questions and decide
whether to improve choice parsing from raw Markdown, adjust DeepSeek prompt
constraints, or downgrade cases where the source itself lacks explicit choices.
